# Machine Learning Pipeline - Student Performance Prediction

## Description
This project predicts a student's final score based on study hours, attendance, and previous performance using a Machine Learning pipeline.

## Features
- Data preprocessing using StandardScaler
- Machine Learning pipeline implementation
- Multiple model comparison
- Evaluation using MSE and R² score
- Best model selection
- Model saving using pickle

## Requirements
- Python 3.x

## Installation

1. Clone repository:
   git clone https://github.com/your-username/ml-pipeline-student

2. Navigate to project folder:
   cd ml-pipeline-student

3. Install dependencies:
   pip install -r requirements.txt

## How to Run

Run the following command:

python main.py

## Output
- Displays model performance (MSE and R²)
- Selects best model
- Saves trained model as model.pkl

## Files
- main.py → Main pipeline script
- utils.py → Helper functions
- data.csv → Dataset
- model.pkl → Saved model (auto-generated)
